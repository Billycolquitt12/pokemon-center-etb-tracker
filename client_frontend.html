Pokémon ETB Tracker

Track and manage Pokémon Center Elite Trainer Boxes

View All Items

This section shows every ETB item currently saved in your database. Use this when you want to quickly see your full item list all at once.

Find One Item

Use this section to search for one specific item by its Item ID. This is helpful when you only want to view one record instead of the full list.

The Item ID is the unique number assigned to each item in the database.

Create New Item

Use this form to add a new ETB to the tracker. Choose a product set from the dropdown so the item is linked to a valid set in your database.

This is the user connected to the item. Leave the default if you are only using one user for your project.

Choose the Pokémon set this item belongs to. Only valid sets from your database appear here.

Enter the full name of the ETB item you want to store.

Choose Yes if the ETB is still sealed, or No if it has been opened.

Update Item

Use this section to change the information for an item that already exists in the database. Choose a valid product set from the dropdown when updating.

Enter the ID of the item you want to update.

This updates which user the item belongs to.

Choose the Pokémon set connected to this item.

Type the new name you want this item to have.

Change the sealed status if needed.

Delete Item

Use this section to permanently remove an item from the database. Be careful, because once deleted, the item record may not be easy to recover.

Enter the ID of the item you want to delete.

Delete is optional in some parts of the project, but this is available if your API supports it.

Create New Product Set

Use this section to add a new Pokémon product set to the database. After creating it, it can be used when creating or updating items.

Enter the name of the Pokémon set.

Enter the year this set was released.

Find One Product Set

Use this section to search for one specific Pokémon set by its Set ID. This is useful when you want details about a single set instead of viewing the full list.

The Set ID is the unique number assigned to each Pokémon set in the database.

View All Product Sets

This section displays every Pokémon product set currently stored in the database. Use it to quickly see all available sets and their release years.

Get All Item Values

This section loads the full price or market value history for all items. It is useful for viewing how item values have been recorded over time.

Get One Item Value

Use this section to look up one specific market value record by its Value ID.

The Value ID is the unique number for a single value history record.

Search Item Values by Item

Use this section to see all recorded value history for one specific item. This helps you track price changes for that item over time.

Enter the Item ID to load all market values connected to that item.

Search Product Sets by Year

This section lets you search for Pokémon product sets based on the year they were released. This is useful if you want to filter sets by release year instead of searching by name.

Enter a year to see all product sets released during that year.
